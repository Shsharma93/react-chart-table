

To run the application, please follow below steps.

Step 1 - open Terminal application.

Step 2 - in the terminal type command to go to the exercise folder. (Using cd command)

Step 3 - now type “nom install” this command and press enter.

Step 4 - once the step 3 process will finished, npm dependencies will be installed.

Step 5 - now run “npm start” command to run the application. 

Step 6 - Step 5 will automatically open the default browser with local server http://localhost:3000/.

Step 7 - Now go to browser and  SPA is ready to use.
